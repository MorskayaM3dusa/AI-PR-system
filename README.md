# Система анализа и улучшения восприятия вашего продукта ИИ-ассистентами (ChatGPT, Claude, Gemini).

## Проблема

В 2025 году пользователи все чаще спрашивают у ИИ-ассистентов "Какой продукт выбрать?", а не ищут в Google. Если ChatGPT не упоминает ваш продукт в ответах - вы теряете клиентов, даже если у вас лучшее решение.

### Пример:
```bash
Пользователь: "Какой инструмент автоматизации выбрать для стартапа?"
ChatGPT: "Рекомендую: 1. Zapier, 2. Make, 3. IFTTT"
Ваш продукт n8n не упоминается → Потеря клиентов
```
## Возможности

- Диагностика восприятия ИИ
- Автоматические запросы к LLM (Mistral AI)
- Анализ упоминаний продукта и конкурентов
- Оценка тональности и аргументации

## Выявление авторитетных источников

- Поиск сайтов, которые цитирует ИИ
- Обнаружение "слепых пятен" (где есть конкуренты, но нет вас)
- Анализ стиля успешного контента

## Генерация ИИ-оптимизированного контента

- Технический контент для ИИ - структурированные данные, метрики
- Контент для внешних площадок - статьи в стиле Dev.to
- Контент для собственных каналов - блоги

## Мониторинг и аналитика

- Дашборд с графиками изменения упоминаний
- Расчет ROI кампании
- Ежедневные автоматические обновления

## Быстрый старт

1. Установка

### Клонирование репозитория
```bash
git clone https://github.com/MorskayaM3dusa/AI-PR-system
cd AI-PR-system
```
### Установка зависимостей
```bash
pip install -r requirements.txt
```

2. Настройка

Отредактируйте config.py:
```bash
TARGET_PRODUCT = "n8n"  # Ваш продукт
COMPETITORS = '["Zapier", "Make", "Microsoft Power Automate", "IFTTT"]'
MISTRAL_API_KEY = "ваш_ключ"  # Или через .env
```

### Запуск консольного интерфейса
```bash
python main.py
```
